Projet phare — PFE

TSP Logistics Manager v2.0

Plateforme prescriptive de pilotage de production
OCP GROUP · TSP HUB 2 · PYTHON / PYQT6

Application desktop intelligente conçue pour transformer le pilotage de production du réactif : simulation Monte Carlo, moteurs d'optimisation multiples, analyse des coûts d'arrêt et assistant conversationnel intégré, au service des équipes SOL/GRA/STA.

MODULE 01
Moteurs d'optimisation
MILP, Génétique, Q-Learning, Greedy, Balance, Avancé — 6 approches comparées.
MODULE 02
Simulation Monte Carlo
Décomposition de Cholesky pour modéliser les corrélations entre variables.
MODULE 03
StopCost Analyzer
7 paliers de réduction du MTTR pour prioriser les actions correctives.
MODULE 04
Pilotage production
Suivi des processus SOL → GRA → STA avec TRS / OEE en temps réel.
MODULE 05
Chatbot NLP
TF-IDF + RandomForest pour répondre aux questions opérationnelles (F1 = 0.847).
MODULE 06
Interface PyQt6
Tableau de bord desktop pensé pour un usage terrain, rapide et lisible.

Stages & missions industrielles

Ingénieur assistant — Architecture décisionnelle intelligente TSP
El Jadida, Casablanca-Settat, Maroc · Sur site
OCP GROUP · STAGE · FÉV. 2026 – MAI 2026 (4 MOIS)
« Conception d'une architecture décisionnelle intelligente pour le pilotage de la performance industrielle du TSP, exploitant l'IA pour anticiper les dysfonctionnements, réduire les arrêts de production et optimiser les flux logistiques. »
Conçu et déployé TSP Logistics Manager v2.0, plateforme desktop (PyQt6/Python) centralisant le pilotage de 20 silos de stockage et 3 flux de production (SOL → GRA → STA). Implémenté 6 algorithmes d'optimisation (Greedy, MILP, Génétique, Q-Learning, Balance, Avancé) et une simulation Monte Carlo pour l'aide à la décision sous incertitude. Intégré un chatbot NLP (TF-IDF + Random Forest, F1 = 0,847) pour l'assistance opérationnelle en langage naturel. Impact : −36% d'arrêts non planifiés, +13,2 points de TRS, disponibilité opérationnelle à 94,5%.
